AI Sign Language → Text Converter

An AI-powered tool that translates Indian Sign Language gestures into text in real-time.

This project detects Indian Sign Language (ISL) gestures using AI. It supports letters A–Z and numbers 0–9. The system uses MediaPipe hand tracking and a Neural Network model to convert signs into text. A hold-to-confirm mechanism (2 seconds at ≥70% confidence) ensures accurate and intentional letter detection.

🖐️

Hand Detection

Uses MediaPipe's 3D dual-hand landmark detection with bone geometry for robust occlusion handling.

🧠

Neural Network

A 316-dim feature feed-forward NN trained on ISL gesture data with Landmark Cutout augmentation.

📝

Text Conversion

Letters are confirmed after a 2-second steady hold, assembling words in real time in the output box.

📸

Image Upload

Alternatively, upload any static image to instantly detect and classify the sign shown in the image.

Tech Stack
  • Python 3 + PyTorch (Neural Network)
  • MediaPipe (3D Hand Landmark Detection)
  • Gradio (REST API backend)
  • HTML5 / CSS3 / JavaScript (Frontend)
  • WebRTC getUserMedia (Live Webcam)
